# 2027 Sakanaction Tour Venues Dataset

> 本文件依 `sakanaction_2027_venue_factcheck.md` 重新分級。  
> 「已確認」代表查證報告明確判定可用；「概略可用」僅適合描述規模；「未驗證」不得當作精確規格；「已否定」不得引用。

## 可信度圖例

- **已確認**：查證報告可對應官方或多方資料。
- **概略可用**：適合寫「約」，不可用於精確座位配置。
- **未驗證**：查無公開來源，需官方規格頁、座席表或 technical rider。
- **已否定**：與官方資料、加總或實測明顯矛盾。

## 使用限制

1. 可對外引用：場館名稱、巡演順序、已確認或概略容量。
2. 可附來源後引用：已確認的個別席位、Sun Dome 建築尺寸、Xebio 面積與高度。
3. 不可直接引用：SS／S／A／B 精確人數、未驗證長寬高度、自行拼接的樓層小計。
4. 若要恢復精確標註，必須先取得官方座席配置或主辦方 technical rider。
